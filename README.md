# Breast Cancer Prediction
Data Science Capstone Project by Anagha Deshpande, Thanya Mysore Santhosh, and Melissa Rejuan.

Our primary data source is the breast cancer METABRIC database. It is publicly available on the Cbioportal for Cancer Genomics. We retrieved a csv download from Kaggle: https://www.kaggle.com/datasets/gunesevitan/breast-cancer-metabric.
